# Dashboard RRHH

Proyecto personal de Business Intelligence orientado al análisis 
de recursos humanos y toma de decisiones sobre el capital humano 
de una organización.

## Descripción

Dashboard interactivo de 3 páginas construido sobre un dataset 
de 194 empleados con información de departamento, posición, 
sueldo, evaluación de desempeño y datos demográficos.

## Dataset

- **Fuente:** Dataset ficticio
- **Registros:** 194 empleados
- **Columnas:** ID Empleado, Nombre, Estado, Fecha de Nacimiento, 
  Género, Departamento, Posición, Nombre Jefe
- **Geografía:** Estados Unidos (California, Texas, Massachusetts)

## Tecnologías utilizadas

- Power BI Desktop
- Power Query (transformación de datos)
- Git / GitHub

## Páginas del dashboard

**Total colaboradores**
Distribución de los 194 empleados por grupo de edad, rango 
salarial y nivel de evaluación. Incluye mapa geográfico por 
estado y tabla detallada por empleado.

**Análisis de sueldo**
Gasto total en sueldos por departamento y análisis de correlación 
entre edad y sueldo. Permite identificar dónde se concentra 
el mayor costo de personal.

**Evaluación de desempeño**
Ranking de jefes por evaluación promedio, comparación por 
departamento y análisis por género. Permite identificar 
qué equipos y líderes tienen mejor rendimiento.

## Decisiones que permite tomar

- ¿En qué departamento hay mayor concentración de personal?
- ¿Existe correlación entre edad y nivel salarial?
- ¿Qué jefes tienen los equipos mejor evaluados?
- ¿Hay diferencias de desempeño entre géneros?
- ¿Dónde están geográficamente concentrados los empleados?
