# Power BI Projects

Repositorio de proyectos personales de Business Intelligence 
desarrollados en Power BI, orientados al análisis de datos 
y toma de decisiones.

## Proyectos

### 📊 RRHH - Recursos Humanos
Análisis de dotación, estructura salarial y evaluación de 
desempeño sobre 194 empleados distribuidos en 6 departamentos.

**Stack:** Power BI · Power Query · DAX

[Ver proyecto](./rrhh/)

---

### 💰 Finanzas Personales
Análisis de ingresos, gastos y tendencias financieras personales.

**Stack:** Power BI · Power Query · DAX

[Ver proyecto](./finanzas-personales/)

---

### 📞 Customer Experience - NPS
Análisis integral del servicio de atención al cliente: NPS por 
negocio, evolución semanal y ranking de rendimiento por agente.

**Stack:** Power BI · Power Query · DAX

[Ver proyecto](./customer-experience/)

## Tecnologías

- Power BI Desktop
- Power Query (ETL y transformación de datos)
- DAX (medidas y columnas calculadas)
- Git / GitHub

## Sobre este repositorio

Cada carpeta contiene el archivo `.pbix` del proyecto y su 
propio README con la descripción detallada, preguntas de 
negocio, hallazgos y stack utilizado.
